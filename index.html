GW2 Personal Assistant Overlay

A stand-alone Guild Wars 2 informational overlay.



World Events Tracker

Tracks active and inactive world events, and allows a user to keep track of which events have already been completed for the day.

World Event Notifications

Pop-ups that appear when a world event is about to start.

WvW Tracker

Displays all objectives in the player's WvW map, including owner, distance to the objective, and RI timer value.

WvW Notifications

Pop-ups that appear when an objective has been captured in one of the WvW maps.

Zone Completion Assistant

Displays real-time distance and direction information for zone completion locations with several options for keeping track of unlocked locations.

Dungeon Tracker

Keeps track of what dungeon paths have been completed for the day, and provides information about each dungeon and path.

Trading Post Price Notifications

Pop-ups that appear when an item has reached a certain highest Buy Order or lowest Sale Listing in the Trading Post.

Trading Post Calculator

Calculates profit, taxes, and break-even prices for buying/selling items on the in-game Trading Post.

Teamspeak Overlay

Shows when someone talks, sends a chat, or enters/exits the active channel. Provides channel switching and messaging.

Web Browser

Provides a way to look up guides, videos, or any other useful website.

Download

GW2 PAO comes packaged as an installer or zip archive.

For the zip-file download, two versions are available, one with all features included and another excluding the browser. A browserless version is provided for those who may be concerned about the security of the Awesomium browser.

The installer provides an option to include or exclude installation of the browser from the application.

To install, either download and run the installer, or just download the zip,
extract to any location, and run GW2PAO.exe

FAQ

Is this safe to use?

This software does not include any hacks or cheats, and makes use of official APIs built into the game and built by ArenaNet. Using this software is in no way different than using other GW2 informational websites, event trackers, and overlays.

Some additional links:
Chris Cleary's (ArenaNet employee) post regarding version 1.0 of this overlay
ArenaNet's forum post regarding another GW2 overlay utility
ArenaNet's official policy on 3rd-Party Programs

Does this work while playing the game in Fullscreen?

No. Due to some technical reasons for how Fullscreen games take over the monitor, the overlay does not work when the game is running in fullscreen (at this time). The game must be running in Windowed or Windowed Fullscreen for it to properly overlay on top of the game. It's possible that this may be added in the future.

Can I run this on my Mac?

No. Mac OS is not supported at this time.

I'm having an issue with the overlay. How can I report it?

Issues can be reported at the Github Issues Page, or at the GW2 PAO Forums.
